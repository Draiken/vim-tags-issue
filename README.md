# Instructions

```
   make

   make run
```

Once in there, open vim with

```
   vim -u .vimrc b.rb
```

Then install `coc-solargraph`

```
   :CocInstall coc-solargraph
```

Open the file again, and try to navigate to `A` with `Ctrl+]`

To see the working version, use

```
   vim -u .vimrc-working b.rb
```
